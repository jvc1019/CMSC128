How to use ZEST 

1.) Run setup.cmd. This will open Command line and unpack all necessary files
needed to run Zest. A shortcut for Zest will appear in the Desktop

2.) Zest can be opened by running the Zest shortcut in the Desktop,
Startup Menu or the Zest.cmd found inside the Zest folder in Local Disk C:

3.) Zest can be uninstalled in the Zest folder in Local Disk C: by
running Uninstall.cmd There is an uninstaller shortcut that can also 
be found in the startup menu.
